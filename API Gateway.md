# API Gateway

[Amazon API Gateway](https://docs.aws.amazon.com/ko_kr/apigateway/latest/developerguide/welcome.html)는 컨테이너식 서버리스 워크로드, 웹 애플리케이션을 지원하는 완전 관리형 API 관리 서비스입니다.트래픽 관리, CORS 지원, 권한 부여 및 액세스 제어, 제한, 모니터링 및 API 버전 관리 작업 등의 기능을 제공합니다.

애플리케이션이 뒤쪽 백엔스 서비스에 액세스할 수 있는 `정문` 역할을 합니다.주로 Lambda를 연결하여 안정적인 트래픽 관리 및 제어를 담당하도록 구성합니다.
